# Sistem-Informasi-Rental-Mobil
Tugas besar Pemrograman Web 1

### Teknologi yang digunakan
* HTML + CSS + PHP + JS
* Jquery
* Bootstrap + Font Awesome
* Datatables
* Mysql

### Fitur Menu
* Login
* CRUD Customer
* CRUD Mobil (merk, mobil, harga)
* CRUD Peminjaman
* CRUD Pengembalian
